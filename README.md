# CPSC 444 App

1. Clone repository:

```bash
git clone https://github.com/farazht/cpsc444.git
cd cpsc444
```

2. Install dependencies:

```bash
npm install
```

3. Start Expo:

```bash
npx expo start
```

4. Open the app:

- Type `w` for web
- Type `i` for iOS simulator (requires Xcode)
- Type `a` for Android emulator (requires Android Studio)
